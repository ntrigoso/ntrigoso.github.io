List of HTML tags and classes that can be used:

  • <'b'>
    • bold tag will be used for highlighting important information that students should pay attention to
  • <'a'>
    • for links that students can use for teacher information and assignment information
  • <'ul'>
    • used for unordered lists that list the topics or concepts that will be taught for students
  • <'i'>
    • to make a text italic that will be used in when writing the teachers' names and contact information
  • <'img'>
    • used for putting different images within the website and formatting them in any way
  • <'ol'>
    • used for ordered lists that will be used for listing the order in which concepts will be taught in each class
  • <'br'>
    • used for creating breaks within the website and putting more space between differnt sections
  • <'div'>
    • will be used for dividing each section within the website, so that the website looks nicer and pleasant to look at
  • .header class
    • will be used for formatting specifically the header section of the website such as the header's font, font-size, color, background-color, etc...
  • .column class
    • will be used for formatting the columns that will be used within the website so that each column looks similar to one another
  • <'span'>
    • used for styling part of a text
  • .row class
    • will be used for formatting the row sections within the website
